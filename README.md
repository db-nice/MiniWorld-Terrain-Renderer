

### MiniWorld 外部渲染与地形转换器

### 项目概述
这是一个专为**迷你世界**游戏数据处理的**第三方Blender插件**。可将导出的Lua地形数据（`x,y,z,block_id` CSV格式）转换为Blender中的3D模型，主要用于地形复刻、矿物分布映射和方块结构重建。

⚠️ **当前处于开发中**  
目前仅实现**基础地形生成**，不包含细节结构或实体渲染。本工具作为外部渲染和地形分析的基础框架。

### 核心功能
- 解析迷你世界导出的Lua数据（CSV格式：`x,y,z,block_id`）
- 使用 `image_str.txt`（迷你世界内置CSV映射表）将方块ID映射到纹理
- 在Blender中生成方块级精度的3D地形网格
- 支持矿物/矿石分布可视化
- 基础方块复制和结构克隆

### 数据格式示例
```
-1,0,0,399
-1,3,1,990
-4,0,0,1062
```
坐标代表游戏内位置，ID对应方块类型。
